# Instalación

Para instalar las dependencias necesarias para ejecutar el proyecto en su ordenador, siga los siguientes pasos:

1. Vaya al directorio raíz del proyecto:
    <pre><code>C:\Users\tuuser\Desktop\GaktrackAPI></code></pre>
2. En su consola, ejecute el siguiente comando:
    <pre><code>npm install</code></pre>
